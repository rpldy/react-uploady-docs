## Package

**_[@rpldy/uploady](../../../packages/rpldy-uploady)_**

## Installation

```bash npm2yarn
npm install @rpldy/uploady
```

## Details

```typescript
type useUploady: () => UploadyContextType;
```

- See [UploadyContextType](../../types#uploadyContextType)

Shortcut hook to get the [Uploady Context](#context) instance

:::info
alias for `useUploadyContext`
:::

:::warning
Will throw in case used outside of Uploady render tree
:::

## Example

```javascript
import { useUploady } from "@rpldy/uploady";

const MyComponent = () => {
    const uploady = useUploady();

    const onClick = () => {
        uploady.showFileUpload();
    }

    //...
};

const App = () => (
    <Uploady destination={{ ... }}>
        <MyComponent/>
    </Uploady>
);
```
