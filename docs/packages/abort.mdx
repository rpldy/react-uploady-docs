## Installation

```bash npm2yarn
npm install @rpldy/abort
```

## Details

Adds the capability to abort/cancel running & pending uploads
Currently, this is an internal package the uploader consumes.

:::info
In the future it will be possible to use the Uploader without this package when abort functionality isn't required
:::
